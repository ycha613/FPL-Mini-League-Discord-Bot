# FPL-Mini-League-Discord-Bot
Discord bot that provides updates on transfers, point totals, and captains for a mini league.


## Installation

**Prerequisites**
* Node.js v18 +
* npm


**Setup**
Clone the repository
```shell
git clone https://github.com/ycha613/FPL-Mini-League-Discord-Bot.git
cd fpl-mini-league-discord-bot
```

Install dependencies
```shell
npm install
```


## Configuration

Create a file ".env" in the root directory, this file contains variable settings.
Go to the discord developer token to setup your bot and get a token, copy it in this file.
```shell
DISCORD_TOKEN=your token from the discord developer portal
```

## Execution

**Development**
Running the bot with automatic reload on code changes
```shell
npm run dev
```

**Production**
Build the typescript code and and run the compiled javascript
```shell
npm run build
npm start
```
